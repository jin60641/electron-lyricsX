# electron-lyricsX

Lyrics displaying app for windows and macOS.

LyricsX for iOS and lyricsx-cli for Linux is in early development.

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
